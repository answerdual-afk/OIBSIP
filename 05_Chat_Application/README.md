# Chat Application

A beginner-level text-based chat application built using Python.

## Features
- Two users can exchange messages
- Custom usernames
- Displays timestamps for messages
- Saves chat history to a text file
- Exit chat using "bye"

## Technologies Used
- Python
- datetime module

## Files Included
- chat_application.py → Main program
- chat_history.txt → Stores chat messages

## How to Run

1. Run the program:

python chat_application.py

2. Enter usernames

3. Start chatting

4. Type "bye" to end the chat
